# FTP-клиент

## Описание проекта

Простой FTP-клиент с интерактивным консольным интерфейсом, сделанным на Python.

## Возможности

- Подключение к FTP-серверам
- Просмотр файлов и директорий
- Скачивание файлов с сервера
- Загрузка файлов на сервер
- Создание новых директорий
- Удаление файлов (с подтверждением)
- Навигация по директориям
- Отображение текущей директории

## Требования
- Python 3.6 или выше
- Доступ к интернету (для подключения к FTP-серверам)

## Установка зависимостей

Проект использует только стандартные библиотеки Python
